Make it Collaborate
I sketch wireframes and make prototypes.
Talking through tactile designs existing in the
browser is worth its weight. Design tools only
carry you so far; the rest should be realized
with a link my team can rally around. 		Good design is not created in a vacuum but
rather in a shared space. It must be

in my design process and create a
collaborative environment that welcomes and encourages feedback.
